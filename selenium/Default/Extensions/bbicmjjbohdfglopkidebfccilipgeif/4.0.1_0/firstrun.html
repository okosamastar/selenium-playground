FAQ

How do I report a bug in Validity?

If you think you've found a bug in Validity, please raise an issue on the project's GitHub page at https://github.com/renyard/validity/issues.

How to I request a new feature in Validity?

Please raise an issue on the project's GitHub page.

How can I support the development of Validity?

There are several ways that you can contribute to the continued development of Validity:

  • If you like the extension, please leave a review on the Chrome Web Store. If you've encountered any problems or would like to make a feature request, please raise an issue on the project's GitHub page.
  • Code contributions are also always welcome and appreciated for bug fixes and new features. For a bug fix, please raise a pull request.
