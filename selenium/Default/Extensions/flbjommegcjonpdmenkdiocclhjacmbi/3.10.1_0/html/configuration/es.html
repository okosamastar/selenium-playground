Jerarquía de encabezados

Cuando la opción "considerar secciones ocultas" está activa, el análisis puede impactar en el rendimiento cuando la página a revisar es muy larga y compleja.

Actualmente no hay navegadores ni tecnología asistiva con soporte nativo para el algoritmo W3C Outline. Por ese motivo, no se puede confiar plenamente en él para transmitir la estructura de los documentos a los usuarios y usuaria. Los autores de contenido deberçian seguir utilizando los niveles de encabezado (1 to 6) para hacerlo (HTML 5.2 Creating an outline)
